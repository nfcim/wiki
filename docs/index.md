# 概述

本站将围绕 NFC 和智能卡相关的技术，介绍有关技术、协议和标准。主要内容包括：

* NFC 技术
* [智能卡手册](/cards/)：介绍常用卡片的读取方式和数据存储结构
* 常用密码学算法手册
